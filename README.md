# sellgar.shop.service

Сервис магазинов и каналов продаж Sellgar.

Планируемая зона ответственности:

- магазины/витрины/каналы продаж;
- настройки магазина;
- будущие адреса, режимы работы, зоны обслуживания;
- связи, необходимые store service для публикации продаваемых товаров в конкретном магазине.

Каталог товаров и продаваемые варианты этому сервису не принадлежат.
